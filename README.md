# Namesgenerator

Port of Docker 0.7.x names generator written in Ruby for fun!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'namesgenerator'
```

Or

```shell
$ gem install namesgenerator
```

## Usage

```ruby
Namesgenerator.random_name
# => admiring_wozniak
```
